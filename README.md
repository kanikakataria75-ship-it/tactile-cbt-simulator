# 📊 Multi-Exam CBT Simulator

A premium, widescreen 3D glassmorphic Computer Based Test (CBT) Simulator engine designed for comprehensive exam preparation and real-time assessment tracking. Built using a robust Flask backend and an optimized Vanilla JS frontend interface.

## ✨ Key Features
* **3D Glassmorphic UI:** A modern, high-end widescreen interface designed for focused, distraction-free testing.
* **Hybrid Parsing Engine:** Efficiently processes dynamic question formats, accommodating both standard text structures and rich-layout PDFs.
* **Smart Input Rendering:** Seamlessly renders Multiple Choice Questions (MCQs), multi-select prompts, and specialized Numerical/Integer-type response matrices.
* **Advanced Analytics:** Features per-question time tracking, automated error logs, and a specialized dashboard that calculates performance insights.

## 🛠️ Tech Stack
* **Backend:** Python / Flask
* **Frontend:** Vanilla JavaScript / Tailwind CSS (Glassmorphism layer)
* **Data Storage:** Structured JSON Session states
